Everything you need

From mod management to one-click modpack installation — AnonLauncher handles it all.

Multi-Instance

Create, clone, export, and import Minecraft instances. Keep Vanilla, Fabric, and modpacks separate.

Fabric + NeoForge

Full support for Vanilla, Fabric, and NeoForge. Fabric auto-downloads API, NeoForge has a one-click installer.

Mod Management

Search, install, update, and remove mods with auto-dependency resolution. Batch updates with snapshots.

Modpack Install

One-click install from Modrinth (.mrpack). SHA1 verified downloads with live progress tracking.

Auto Java

Automatic Java download from Adoptium API. Supports Java 8 through 25 — no manual setup needed.

Snapshots

Full or metadata-only backups. Perfect for safe mod updates — restore instantly if something breaks.

Resource Packs & Shaders

Browse and install resource packs and shaderpacks directly from Modrinth. Auto-installs Iris + Sodium.

Game Insights

Screenshot gallery, disk usage stats, mod counts, and quick-access links to saves and screenshots.
